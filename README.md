# Tomorrowland-Scheduler

Mobile-first Tomorrowland schedule planner MVP built with Expo React Native and TypeScript.

## Live preview on GitHub Pages

This repo includes a static browser preview in `docs/index.html` and a GitHub Pages workflow in `.github/workflows/pages.yml`.

To publish it live from GitHub:

1. Push the branch containing this commit to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings → Pages**.
4. Under **Build and deployment → Source**, choose **GitHub Actions**.
5. Run the **Deploy live preview** workflow, or push to `main`.
6. Open the Pages URL shown by the workflow deployment summary.

The deployed preview serves `docs/index.html` and links to `docs/ui-preview.svg`.

## Local Expo development

```bash
npm install
npm run start
```
